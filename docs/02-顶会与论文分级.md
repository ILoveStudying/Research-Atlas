# 顶会与论文分级

## 分级目的

论文分级用于决定阅读和复现优先级，不用于简单否定论文价值。Survey、benchmark、产业系统和高质量 arXiv 可以单独标记。

## 推荐分级

| 分级 | 含义 | 处理策略 |
|---|---|---|
| A1 | 核心顶会主会论文 | 优先精读、消化到 wiki、跟踪代码 |
| A2 | 强相关会议、顶刊、领域顶会 | 选择性精读，纳入对比 |
| B | 高质量 workshop、系统论文、代表性 arXiv | 作为方向补充和实现参考 |
| S | Survey、benchmark、tutorial、产业系统 | 单独跟踪，常用于建立研究地图 |

## 常见 A1 会议示例

NeurIPS、ICML、ICLR、CVPR、ICCV、ECCV、KDD、WWW、SIGIR、ACL、EMNLP、AAAI、IJCAI、ICDE。

## 证据强度

每条论文记录建议标注证据强度：

- 官方论文
- 会议页面
- arXiv
- 作者主页
- 二手摘要
