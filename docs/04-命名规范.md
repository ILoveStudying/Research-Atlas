# 命名规范

## slug

目录名使用小写英文 slug，单词之间用连字符连接。

示例：

```text
spatiotemporal-foundation-models
representation-and-tokenization
st-data-tokenization
```

## 中文名

中文名写在 `README.md` 和对应 `*.yml` 元数据里，便于 GitHub 和 Obsidian 阅读。

## 论文 id

论文记录文件名建议使用：

```text
<年份>-<短标题>.yml
```

示例：

```text
2025-bigcity.yml
2024-urban-dit.yml
```

## 代码 id

代码记录文件名建议使用项目名或论文短名：

```text
bigcity.yml
urban-dit.yml
```
