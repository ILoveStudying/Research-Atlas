# 代码层

- `repos/`：代码仓库登记。
- `github/`：通过 Git submodule 下载的外部代码仓库。
- `configs/`：关键配置快照。
- `patches/`：本地修改补丁。
- `runs/`：复现实验记录。
- `issues/`：依赖、数据、显存和复现问题。
