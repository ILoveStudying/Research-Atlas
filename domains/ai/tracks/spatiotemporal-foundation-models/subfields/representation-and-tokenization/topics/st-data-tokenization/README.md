# 时空数据 Token 化

## 研究问题

这个 topic 系统追踪真实世界时空数据如何被 token 化、序列化、离散化或 prompt 化，并比较这些表示如何服务大模型、世界模型、预测模型和工具调用。

## 关键问题

1. 不同论文如何定义时空 token？
2. token 中包含哪些空间、时间、拓扑、语义、事件和物理字段？
3. tokenization 与下游任务之间是什么关系？
4. 哪些方法可以跨城市、跨模态、跨任务泛化？
5. 代码实现中哪些配置决定了 token 表示质量？
